# Turing Summer Program
Documents, code examples, and tutorials for the CICS Turing Summer program.

## Online editor for micropython on the micro:bit:
https://python.microbit.org/v/3
The website has a micro:bit emulator for practice without the hardware.

## Follow the examples starting with 8.1, 8.2, etc.
Connect the micro:bit to your computer with a USB micro cable.
Write the code, then upload it. The website walks you through the process.
